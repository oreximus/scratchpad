# Class Notes

## NetBIOS(NBNS) in Windows:

- It's a protocol works in the LAN for only searching or communicating with PCs, via their
  hostname only.

- commands:

- same pc entries:

```
nbtstat -n
```

- resolving the cache names:

```
nbtstat -c
```

- removing the cache

```
nbtstat -R
```

- session simple view:

```
nbtstat -s
```

- session detailed view:

```
nbtstat -S
```

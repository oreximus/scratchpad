# Common Linux Useful Commands:

## for copying STDOUT directly to clipboard:

- for checking the XDG_SESSION_TYPE:

```
echo $XDG_SESSION_TYPE                                                                                                       ✔ 
```

- for `xorg` Display Manager:

**using xsel**:

```
command | xsel --clipboard
```

**using xclip**:

```
command | xsel --clipboard
```
